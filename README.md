Company Celebrations Web Part

Wedpart is designed to display the users birthdays, anniversaries and certification celebrations based on a sharepoint list. The birthday will automatically display the year of the users, the anniversary wil display the years at work, and the certification will display the congratulation for a new certification. The certification expiration date is to hide the ceritication after a certain period of time.

  - Data Source: Employee Celebrations list selected
  - Display Mode: Grid View
  - Show Events: This Month
  - Show Images: Off
  - Show Title: Off
  - Center Content: Off
  - Color pickers for Birthday and Anniversary colors
  - Visibility settings
